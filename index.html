C/C++ Math functions
pow abs floor ceil exp
pow

Calculates power of ab.

Both a and b are floating point double precision numbers

double a = pow(2, 4); // Stores 16 (2 raised to 4) in a
abs

Calculates absolute value of a.

If a is negative, returns -a

double a = abs(-2); // Returns 2

If a is positive, returns a

double b = abs(2); // Returns 2
floor

Calculates floor value of a.

a is a double precision floating point number.

  • If x is between a and a + 1, returns a
  • If x is a, returns a
  • If x is a + 1, returns a + 1
double a = floor(3.56); // Stores 3 in a
ceil

Calculates ceiling value of a

a is a double precision floating point number.

  • If x is between a and a + 1, returns a + 1
  • If x is a, returns a
  • If x is a + 1, returns a + 1
double a = ceil(3.56); // Stores 4 in a
exp

Calculates ex

a is a double precision floating point number.

double a = exp(2.3); // Stores 9.974 in a
